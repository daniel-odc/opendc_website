Conselhos acerca do uso de dados telefónicos ou de GPS

O que a OpenDC pode fazer por si

O fundador da OpenDC desenvolveu competências no uso de dados de telemóvel e GPS para entender o movimento de pessoas. Esta competência envolve a recolha de dados, armazenamento, limpeza, processos de filtração que usam diferentes ferramentas matemáticas, escalamentos, correcção de enviesamentos, agregação, análise e visualização.

Como é que este serviço pode ajudar a sua organização

A OpenDC pode assisti-lo com dados de GPS e redes telefónicas se estiver a trabalhar para

  • Organizações não governamentais (ONG's) e queira entender como é que dados GSM podem substituir pesquisas tradicionais nos paises em desenvolvimento.
  • Operadoras de Transporte e esteja a considerar o uso de dados GSM ou GPS para compreender melhor não só o uso da sua rede mas também a potencial demanda.
  • Programas de Investigação que queiram feedback com experiência de campo no uso de dados de telemóvel.
  • Consultadorias dispostas a perceber as vantagens e limitações de cada técnica de recolha de dados de telemóvel.
  • Quaisquer negócios que tenham ou possam ter acesso a dados de telemóvel (GSM, GPS, Bluetooth ou Wifi) e queira entender como é que estes dados podem ser usados, quais as vantagens e limitações, quais as desvantagens e como podem ser evitadas.
