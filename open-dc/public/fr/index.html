fr - About us

fr - OpenDC is a start-up specialised in data analysis and mobility studies. OpenDC mainly works with open data sets and open source languages.

fr - OpenDC helps businesses and public bodies find open data that can help them get new insights, be more efficient or develop new products. We help find, download, store, analyse and visualise the data. We use open source tools and can provide our client with the source code and tools developed to deliver our product. OpenDC has an extensive experience in the analysis of mobility data such as mobility survey data, GPS and mobile phone data but is not limited to it.
