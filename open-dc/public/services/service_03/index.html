QGIS Plugin Development

What OpenDC can do

OpenDC can develop for you custom QGIS plugins under the QGIS API. Note that any QGIS plugin that is publicly released has to be open-source. You can however use your plugin internally if you do not want to release the code.

How this service can help your organisation

You might be interested in a custom QGIS plugin if you are working for a

  • Non Governmental Organisation (NGO) with interesting geographical datasets that you want to make available to the GIS community in order to enable the wide analysis of this data set.
  • Transport operator and want to create your free and open-source transport modelling tool under the QGIS framework. Your strategy could be to use this tool as a transparent and open source framework and let the world GIS community improve it.
  • Research programme and want to be able to do some custom mapping processing that is not possible with the existing GIS tools.
  • Consultancy that uses QGIS every day and would like to automate specific tasks.
  • Urban planner who would like to automate the production of maps.
  • Non Profit organisation and want to give people around the world specific mapping tools related to your activity.
  • Any business that uses GIS via the user interface and would like to be more productive, generate less errors and develop new techniques and products by automating GIS tasks.
